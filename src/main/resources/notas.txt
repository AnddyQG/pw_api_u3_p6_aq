Nivel 1:
Esto es el path{
Donde va el protocolo ip puerto se mantiene(https://localhost:8080/)
Nombre o informacion de la API (API/v1/Matricula)
El nombre Matricula debe estar declarado en singular y sin el cion como si fuera accion
Debe ser un dominio

Nombre del servicio del API debe ser un solo nombre en prural
(/estudiantes)
tambien se puede(/estudiantes/{id})
}

Declaracion adecuada del path + el verbo  -> es un END POINT

/estudiantes/{id} + GET

/estudiantes + GET

/estudiantes + Post
viene acompañado con su body

/estudiantes/{id} + Put
viene acompañado con su body

/estudiantes/{id} + Patch
 viene con su body

 /estudiantes  + Delete (Borra todos los estudiantes)
 /estudiantes/{id} + Delete (Borra un estudiante en especifico )
